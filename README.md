Pour exécuter mon programme :
faire dans l'ordre :
"mvn compile", puis
"mvn package", puis
"java -jar target/tp_qualite-1.0.jar".

Enfin, pour générer la javadoc, il faut entrer la commande :
"mvn javadoc:javadoc"
Les fichiers pour voir la javadoc seront dans "target\site\apidocs".
